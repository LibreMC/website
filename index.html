LibreMC Earth server

LibreMC is a new Minecraft Earth server for 1.21.5 Java edition. It features:

  • A 1:750 Earth map
  • A Gold-based economy
  • Town and nation building
  • No war

This server allows you to build towns and nations on a real Earth map. The focus is building and expanding towns and nations, giving you a place for geopolitical roleplay and building. There is no war and PvP is restricted to arenas, so you don't have to worry about losing anything you've worked hard for.

The gameplay is designed to be simple and not overly complicated, it's meant to be as close to vanilla Minecraft as possible.
